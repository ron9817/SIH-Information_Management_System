# SIH-Information_Management_System
This project was developed as a part Smart India Hackathon 19<br>
## Description
Ministry of statistics requests data from various organizations and have to keep track of all the replies. IMS will help ministry manage, analyse all the replies and also have contents of replies summarized and consolidated in one document.

** The code uploaded here only has web app code the internal working and logic code has not been uploaded.
